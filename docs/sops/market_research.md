# Market Research

This SOP outlines the steps to perform market research on a specific topic.

1. **Define Research Goals**
   Understand what the user wants to research. Identify key questions to answer.

2. **Search for Information**
   Use search tools (e.g., Google, Gemini) to find relevant articles, reports, and data.
   Focus on recent and authoritative sources.

3. **Analyze Findings**
   Summarize the key points from the search results. Identify trends, opportunities, and challenges.

4. **Compile Report**
   Create a structured report with the findings. Include an executive summary and detailed sections.
   Save the report to a file (e.g., `market_research_report.md`).
