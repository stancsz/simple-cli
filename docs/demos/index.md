# Simple CLI Demos

Explore what the **Simple CLI** agent can do. We provide specialized agent personas for different domains.

## Available Demos

### 1. [Oil & Gas Data Analyst](./oil_gas.md)
*   **Role**: Senior Production Engineer.
*   **Skills**: Python Analytics (pandas/scipy), Visualization, Git Integration.
*   **Use Case**: Analyze production well data, plot decline curves, and generate daily reports in Microsoft Teams.

### 2. [General Purpose Developer](./developer.md)
*   **Role**: Autonomous Software Engineer.
*   **Skills**: Bug Fixing, TDD, Feature Implementation, PR Creation.
*   **Use Case**: Receive bug reports via Slack/GitHub, reproduce them with tests, fix the code, and open a Pull Request.
