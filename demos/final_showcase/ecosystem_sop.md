# Ecosystem Federation SOP

## 1. Discovery and Spawning
- Spawn a rapid_prototyping child agency.
- Spawn a creative_design child agency.

## 2. Onboarding
- Read company contexts for Client A and Client B.
- Load context into the respective child agencies' memory.

## 3. Project Federation
- Delegate "Build UI" task to creative_design agency.
- Delegate "Build Backend Prototype" task to rapid_prototyping agency.

## 4. Ghost Mode Operations
- Run a simulated daily check-in script across agencies to report task completion statuses.
- Write updates to HR / Operations logging system.

## 5. Economic & Business Cycle
- Record Xero invoices for services rendered.
- Push HubSpot CRM updates indicating project transition to completion.

## 6. Corporate Consciousness Review
- Gather the simulated Board.
- Provide strategic summary of the ecosystem's performance.
- Approve next steps based on financial status and system efficiency feedback.
